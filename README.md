# Assortment - C Programming Project

## Overview

This project is developed in C language to demonstrate the use of 1D and 2D arrays along with loops and conditional statements. The program performs different array operations such as finding negative elements, finding the largest element in a 2D array, generating the transpose of a matrix, and calculating the sum of elements in a specific row and column.

## Features

* Find negative elements in a 1D array.
* Find the largest element in a 2D array.
* Display the transpose of a 2D matrix.
* Calculate the sum of elements of a selected row.
* Calculate the sum of elements of a selected column.

## Technologies Used

* C Programming Language
* GCC Compiler
* Visual Studio Code

## Learning Outcomes

* Understanding of 1D and 2D arrays.
* Use of nested loops.
* Matrix operations.
* Conditional statements and array traversal.
* Problem-solving using C programming.
